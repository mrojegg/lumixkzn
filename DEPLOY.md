# Deploy Checklist

## 1. Репозиторий

- Создать репозиторий на GitHub или GitLab
- Залить содержимое этой папки в ветку `main`

## 2. Сервер

Рекомендуемый путь для этого проекта:

- VPS на Ubuntu
- Nginx
- SSL через Let's Encrypt

## 3. Файлы сайта

Папка сайта на сервере:

`/var/www/lumixkzn-site`

Основной файл:

`index.html`

## 4. Домен

После покупки домена:

- прописать `A` запись для `@` на IP сервера
- прописать `A` запись для `www` на IP сервера
- либо сделать редирект `www -> без www`

## 5. Nginx

Базовая логика:

- `server_name yourdomain.ru www.yourdomain.ru`
- `root /var/www/lumixkzn-site`
- `index index.html`

## 6. SSL

- установить `certbot`
- выпустить сертификат для домена
- включить автообновление

## 7. Финальная проверка

- сайт открывается по `https`
- редиректы работают
- адаптив проверен
- ошибок в консоли нет
- реквизиты и контакты актуальны
